## 产品概述

AI 物料中台是一个企业内部物料标准化管理与申请审批平台。面向超级管理员、物料库管理员、物料管理员、一线员工、审批人员五类角色，支持物料从「类目标准建立 → 物料档案管理 → 申请审批 → 停采停用」的全生命周期。

---

## 整体结构

- **顶部导航（左侧一级菜单 + 顶部横栏）**：左侧固定一级菜单（标准管理、物料管理、申请流程、系统管理）；顶部横栏显示系统名称、用户头像/账号
- **主内容区**：右侧为主工作区，以卡片/表格为主，展示列表数据
- **权限差异**：超级管理员可见所有菜单；普通用户根据角色权限显示对应功能

## 一、标准管理

### 1.1 左侧菜单结构

```
标准管理
 ├─ 类目库管理
 ├─ 类目管理
 ├─ 品名管理
 ├─ 属性管理
 ├─ 品牌管理
```

### 1.2 通用列表页布局

- 页面标题 + 「新增」按钮（右上角）
- 搜索栏（按名称、编码模糊搜索）
- 表格（编号 / 名称 / 编码 / 描述 / 状态 / 操作列）
- 分页控件

### 1.3 各功能模块说明

#### 类目库管理
- 列表字段：编号、类目库名称、类目库编码、描述、管理员、操作
- 操作列：「编辑」「删除」（仅当下无子类目时显示删除）
- 新增弹窗：类目库名称（必填）、类目库编码（自动生成，展示态只读）、描述、管理员下拉

#### 类目管理
- 树形结构展示（一级类目可展开显示子级类目）
- 列表字段：编号、类目名称、类目编码、父级类目、顺序、操作
- 新增类目时，一级类目无父级选择；非一级类目自动选中父级

#### 品名管理
- 列表字段：编号、品名、编码、所属类目、品名单位、操作
- 新增/编辑弹窗：品名（必填）、品名单位、别名、品名定义、描述规范示例、示例图片上传区（最多3张，拖拽/点击上传，预览缩略图）
- 编码、所属类目在编辑态只读

#### 属性管理
- 列表字段：编号、属性名称、属性类型、是否必填、默认值、操作
- 新增/编辑弹窗：属性名称（必填）、属性类型下拉（默认选项）、是否必填开关（默认开启）、可选择项列表（最少1项，支持新增/删除项）、默认选项下拉、填写提示输入框
- **属性日志**入口：在属性管理页面，每行操作列放置「日志」按钮，点击展开日志面板（时间轴样式，显示修改内容、操作人、操作时间）

#### 品牌管理
- 列表字段：编号、品牌名称、品牌编码、描述、Logo、操作
- 新增/编辑弹窗：品牌名称（必填）、品牌编码（自动生成只读）、描述、Logo上传区（点击上传/替换，预览显示）

## 二、物料管理

### 2.1 左侧菜单结构

```
物料管理
 ├─ 物料库管理
 └─ 物料管理
```

### 2.2 物料库管理
- 入口提示：系统总览页面包含「新建物料库」按钮（仅超级管理员可见）
- 列表字段：编号、物料库名称、物料库编码、描述、管理员、操作
- 编码规则提示：拼音首字母前2位大写（如「办公用品」→ GY）

### 2.3 物料管理
- 顶部：类目选择器（支持下拉切换）+ 搜索框 + 「导出」按钮（主内容区上方醒目位置）
- 列表字段：编号、物料名称、物料编码、所属类目、规格型号、计量单位、图片（缩略图）、状态（正常/停采/停用）、操作
- 状态标识：正常→绿色标签；停采→橙色标签；停用→灰色标签
- 操作列：「编辑」「删除」「停采」「停用」按钮（停采物料隐藏停采按钮，停用物料隐藏停用按钮）
- 新增/编辑页面：
  - 左侧：类目选择器（联动显示当前类目下的品名和属性字段）
  - 右侧：根据所选品名动态渲染属性填写表单（必填属性高亮标识）
  - 底部：物料图片上传区（最多3张）、附件上传区
  - 物料编码自动生成，只读显示
- **停采操作反馈**：弹窗提示「停采成功/停采失败」+ 失败原因说明
- **停用操作前置判断**：未停采物料点击停用按钮 → 弹窗提示「请先完成停采流程」

## 三、物料申请流程模块

### 3.1 左侧菜单结构

```
物料申请流程
 ├─ 新增物料类目申请
 ├─ 新增物料编码申请
 ├─ 物料停采申请
 └─ 物料停用申请
```

### 3.2 申请单通用页面结构

- 顶部：申请单状态标签（草稿/审批中/已通过/已驳回）
- 申请信息区块（灰色底分隔）：单据编码（自动生成）、申请人员（自动带出）、所属单位（可编辑）、申请日期（自动获取）
- 审批流程展示区：横向时间轴，显示当前审批节点（高亮当前节点，虚化后续节点）
- 底部：「提交审批」按钮；草稿态显示「保存草稿」按钮

### 3.3 各申请单说明

#### 新增物料类目申请
- 申请单内容：填写一/二/三级类目（联动下拉）、物料名称、品类单位等
- 其他信息区：物料定义（必填）、参考文件上传（必填）、物料图片上传、采购商城链接输入框
- 审批通过后：资产管理部手动在物料库新增类目（系统不自动处理）

#### 新增物料编码申请
- 物料名称自动拼接显示（根据所选类目属性自动生成，不可手动编辑）
- 物料图片区：3张必填（红色星标提示），拖拽上传，支持预览
- 审批完结后：展示最终物料编码、三级类目、物料名称（只读，不可再修改）

#### 物料停采申请
- 停采原因说明输入框（必填，大文本框）
- 附件上传区
- 停采物料明细表格：
  - 「添加物料」按钮 → 打开物料选择弹窗（搜索+列表选择）
  - 列表字段：物料编码、物料名称、规格型号、总库存（默认0排前）、操作（移除）
  - 每行物料需填写停采原因（红色星标）
  - 支持批量移除

#### 物料停用申请
- 前置条件校验：未完成停采的物料不允许在选择列表中出现
- 停用原因说明（必填）
- 停用物料明细表格（结构同停采申请）：从已停采物料列表中选择
- 停用原因（必填，每行）

### 3.4 审批流程可视化

- 横向时间轴节点：申请人提交 → 部门正/副职审批 → 资产管理部审批 → 完结
- 各节点显示：节点名称、审批人姓名、审批时间、审批状态（通过/驳回/待审批）
- 驳回态：驳回节点高亮红色，显示驳回原因气泡

## 四、系统基础管理

### 4.1 左侧菜单结构

```
系统管理
 ├─ 用户管理
 │   └─ 用户列表
 ├─ 角色与权限
 │   ├─ 角色管理列表
 │   └─ 权限配置页
 └─ 系统配置
     ├─ 系统信息配置
     ├─ 原因选项维护
     └─ 审批模式切换
```

### 4.2 用户管理
- 列表字段：编号、用户名、单位、部门、班组、账号归属、操作
- 新增/编辑弹窗：用户名（必填）、密码/确认密码（新增时必填）、账号归属下拉
- 操作：「编辑」「重置密码」「删除」（删除后用户不可访问系统）

### 4.3 角色与权限
- 角色列表（HCM同步数据，仅查看）：编号、角色名称、角色代码、描述、是否启用、角色类型、创建时间
- 角色新增/编辑弹窗：角色名称（必填）、角色代码（必填）、是否启用（开关）、角色类型（必填）、描述
- 角色启用/停用：开关切换，控制权限生效状态
- **角色绑定用户**：弹窗展示该角色下的关联用户列表，支持添加/移除
- **权限配置页**：
  - 左侧：功能目录树（层级展开）
  - 右侧：当前选中目录对应的按钮/接口权限复选框列表
  - 底部：「保存配置」「还原默认」按钮

### 4.4 系统配置

#### 系统信息配置
- 表单布局：系统名称输入框、系统图标上传区
- 提示文字：图标链接目前需技术人员配置

#### 原因选项维护
- 两个区块：「停采原因选项」「停用原因选项」
- 每块：选项列表（可新增/删除）+ 「新增选项」按钮

#### 审批模式切换
- 单选卡片布局：
  - 卡片A（默认选中）：简易审批 → 说明文字：申请提交 → 资产管理部审批（2步）
  - 卡片B：工作流审批 → 说明文字：多节点层级审批（支持配置节点）
- 选中态：卡片加粗边框 + 蓝色背景 + 选中图标

## 五、通用组件

| 组件 | 说明 |
|------|------|
| 表格 | 支持排序、分页；操作列固定右侧；空状态显示占位插图+文字 |
| 弹窗/抽屉 | 标题+关闭按钮+表单内容+底部操作按钮；遮罩层灰透明 |
| 搜索栏 | 左侧标签+右侧输入框；回车触发搜索 |
| 状态标签 | 小圆角矩形：正常(绿)、停采(橙)、停用(灰)、草稿(蓝)、审批中(黄)、已驳回(红) |
| 时间轴 | 横向实线+节点圆点+连接线；已完成节点实心绿色；当前节点脉冲动画；未完成节点空心灰 |
| 上传组件 | 点击/拖拽上传；支持进度条；上传完成后显示缩略图+删除按钮 |
| 级联选择器 | 类目选择（省市区风格）；选中后高亮父子节点 |
| 表单校验 | 必填项红色星标；提交时错误字段下方红色提示文字 |
| 删除确认 | 二次确认弹窗：大号警示图标+文案「删除后数据不可恢复，确定删除？」+ 取消/确认按钮 |

## 六、状态流转图（物料）

```
[正常物料]
  ↓ 停采申请审批通过
[停采物料] ─ 停采物料在采购单中不可选
  ↓ 停用申请审批通过
[停用物料] ─ 完全退出使用
```

- 状态不可逆向：停采→停用，不可跳过
- 删除操作在任何状态均需二次确认
